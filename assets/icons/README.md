# Extension Icons

Icons are required for the Chrome extension. You can:

## Option 1: Generate icons automatically

Use the included icon generator:

```bash
open generate-icons.html
```

Click the "Download" buttons to download the 3 icons (16x16, 48x48, 128x128).

## Option 2: Use your own icons

Create or place your own PNG icons in this folder with the following names:

- `icon16.png` (16x16 pixels)
- `icon48.png` (48x48 pixels)
- `icon128.png` (128x128 pixels)

## Option 3: Temporary icons

To test the extension without icons, Chrome will use a default icon. Icons can be added later.
